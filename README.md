# 🍅 好用的番茄時鐘 (Useful Pomodoro Timer)

一個基於莫蘭迪色系設計的純前端番茄工作法計時器，旨在提供「簡單、專注、好用」的使用者體驗。

## ✨ 特色功能
- **🎨 五大主題切換**：提供經典莫蘭迪、北歐海洋、鼠尾草森林、陶土玫瑰、午夜沉靜五種質感主題，滿足不同氛圍需求。
- **⚙️ 自定義時間**：可自由調整「工作時間」與「休息時間」（預設為 25/5 分鐘），並即時生效。
- **🔊 優雅提示音**：番茄鐘結束時播放清脆優雅的叮聲（Chime），並提供**音效開關**功能。
- **💾 自動儲存 (LocalStorage)**：
  - 自動紀錄已完成的番茄數量。
  - 記住您偏好的主題色彩。
  - 記住您設定的時間間隔與音效開關狀態。
- **🔒 瀏覽器優化**：針對現代瀏覽器（Chrome/Edge）音效播放限制進行優化，確保提示音順利播放。
- **📱 響應式佈局**：完美支援桌機與手機瀏覽，介面簡約無干擾。
- **🗑️ 自定義彈窗**：清除紀錄時使用質感 Modal 確認，而非生硬的瀏覽器對話框。

## 🚀 使用說明
1. **設定時間**：在上方輸入框輸入您想要的工作與休息分鐘數。
2. **開始專注**：點擊「▶️ 開始」按鈕，時鐘將開始倒數。
3. **切換主題**：點擊右上角的五個彩色圓點，即可即時切換不同的配色風格。
4. **管理音效**：透過設定區的開關決定是否開啟結束提示音。
5. **進度追蹤**：下方會顯示累計完成的番茄數，點擊「清除紀錄」可重新計數。

## 🛠 技術實作
- **HTML5**: 語義化結構與 SEO 優化。
- **CSS3**: 使用 CSS Variables 實作主題系統、Flexbox 佈局。
- **JavaScript**: 原生 ES6 實作計時邏輯、狀態管理、跨分頁授權音效播放。

## 📂 檔案結構
- `index.html`: 主頁面結構與配置。
- `style.css`: 包含五大主題的樣式定義與動態效果。
- `script.js`: 核心操作邏輯、音效控制與本地數據持久化。
