# 產生「好用的番茄時鐘」系統提示詞 (Prompt)

這是一份可以用來要求 AI 工程師產生與本系統相同規格的完整提示詞。

---

## 系統需求提示詞

請扮演一名**高階前端工程師兼 UI/UX 設計師**，為我開發一個名為「**好用的番茄時鐘**」的 Web 應用程式。此程式需遵循以下技術規格與設計美學：

### 一、技術限制
1. **檔案結構**：僅包含三個檔案：`index.html`、`style.css`、`script.js`。
2. **無依賴開發**：不准使用任何外部套件（如 React, Vue, jQuery）或 CDN（如 Tailwind, Bootstrap, FontAwesome）。音效需以 Base64 內嵌。
3. **瀏覽器支援**：支援現代瀏覽器 (Chrome, Edge, Firefox)，並考慮音效播放的自動播放限制機制。

### 二、介面設計 (UI/UX)
1. **設計風格**：
   - 採用**莫蘭迪色系 (Morandi Color Palette)**，追求簡約、高級、不干擾的專注感。
   - 所有元素需具備平滑的過渡效果 (Transition) 與隱微的微動畫（如計時中時鐘具備呼吸感）。
   - 介面置中，包含標題、狀態顯示（工作中/休息中）、大型數字時鐘、控制區（開始/暫停/重置）、累計番茄數。
2. **多主題系統**：提供 5 種預設配色，並透過右上角的彩色圓點快速切換：
   - **經典莫蘭迪** (米/橘)
   - **北歐海洋** (灰藍)
   - **鼠尾草森林** (橄欖綠)
   - **陶土玫瑰** (粉橘)
   - **午夜沉靜** (深色模式)

### 三、核心功能需求
1. **自定義計時**：使用者可自行輸入「工作分鐘數」與「休息分鐘數」，設定需即時同步至時鐘顯示。
2. **計時邏輯**：
   - 工作與休息時間自動循環切換。
   - 具備開始、暫停、重置功能，確保計時器不會重複疊加或邏輯錯亂。
3. **音效與提醒**：
   - 計時結束時播放優雅的提示音（建議 Chime 音效，Base64 格式）。
   - 包含一個**音效開關切換器**。
   - 在程式開始時處理瀏覽器對音效的限制，引導使用者點擊解鎖。
4. **數據持久化 (LocalStorage)**：
   - 紀錄已完成的番茄數。
   - 記憶使用者最後選取的主題、自定義時間與音效開關狀態。
5. **自定義對話框 (Modal)**：
   - 點擊「清除紀錄」時，不使用 `window.confirm`，改用符合主題設計的自定義 Modal 進行確認。

### 四、程式品質
- **JavaScript**：結構化撰寫，清楚區分「狀態管理」、「計時邏輯」與「UI 更新」。
- **CSS**：使用 CSS 變數實作主題系統，支援響應式佈局 (Responsive Design)。
- **HTML**：符合語義化與 SEO 基礎配置。
